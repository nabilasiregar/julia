# Julia Mini Projects👩🏽‍💻
This is a personal mini project to learn Julia programming language

## Prerequisite 
Julia 1.9.3

## How to Run
Clone the project
```bash
  git clone https://github.com/nabilasiregar/julia
```
Go to the project directory

```bash
  cd julia
```

Run the file
```bash
  julia <filename>.jl
```
